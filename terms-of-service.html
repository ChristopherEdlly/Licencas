Aceitação dos Termos

Ao acessar e utilizar o Dashboard de Licenças Prêmio da Secretaria de Estado da Fazenda de Sergipe (SEFAZ-SE), você concorda em cumprir e estar vinculado aos seguintes termos e condições de uso.

Se você não concorda com qualquer parte destes termos, não utilize este sistema.

Elegibilidade e Acesso

Usuários Autorizados:
  • Servidores ativos da SEFAZ-SE
  • Com email corporativo @fazenda.se.gov.br válido
  • Autenticados via Microsoft Identity Platform

Responsabilidades do Usuário

  • Uso legítimo: Utilizar o sistema apenas para fins profissionais relacionados às suas atividades na SEFAZ-SE
  • Confidencialidade: Não compartilhar suas credenciais de acesso com terceiros
  • Segurança: Manter suas credenciais seguras e reportar imediatamente qualquer acesso não autorizado
  • Responsabilidade: Você é responsável por todas as atividades realizadas sob sua conta

Uso Apropriado do Sistema

Você PODE:

  • Visualizar cronogramas de licenças-prêmio
  • Gerar relatórios e análises
  • Exportar dados para fins profissionais
  • Usar filtros e ferramentas de análise
  • Salvar preferências e configurações locais

Você NÃO PODE:

  • Compartilhar dados confidenciais fora da organização
  • Usar o sistema para fins pessoais ou comerciais
  • Tentar acessar áreas restritas ou dados não autorizados
  • Fazer engenharia reversa, modificar ou descompilar o código
  • Usar scripts automatizados para scraping de dados
  • Sobrecarregar ou prejudicar o funcionamento do sistema
  • Violar quaisquer leis ou regulamentos aplicáveis
  • Compartilhar credenciais de acesso

Todo o conteúdo, código-fonte, design, gráficos, interface e funcionalidades deste sistema são de propriedade da SEFAZ-SE ou licenciados para uso.

Licenças de Software

  • Código próprio: Desenvolvido pela SEFAZ-SE para uso interno
  • Bibliotecas de terceiros: Sujeitas às suas respectivas licenças (MIT, Apache, etc.)
  • Microsoft Platform: Uso sob Microsoft Platform Agreement

Proteção de Dados

O tratamento de dados pessoais através deste sistema segue:

  • LGPD (Lei Geral de Proteção de Dados - Lei nº 13.709/2018)
  • Política de Privacidade disponível em: Política de Privacidade
  • Regulamentos internos da SEFAZ-SE sobre proteção de dados

Seus Compromissos

  • Tratar dados de outros servidores com confidencialidade
  • Não divulgar informações pessoais de terceiros
  • Usar dados apenas para finalidades autorizadas
  • Reportar qualquer vazamento ou incidente de segurança

O sistema é fornecido "como está". Não garantimos:

  • Disponibilidade ininterrupta ou livre de erros
  • Correção de todos os bugs ou falhas
  • Compatibilidade com todos os navegadores e dispositivos
  • Manutenção indefinida do sistema
Manutenções:

O sistema pode passar por manutenções programadas ou emergenciais que resultarão em indisponibilidade temporária. Sempre que possível, avisaremos com antecedência.

Limitação de Responsabilidade

A SEFAZ-SE não será responsável por:

  • Danos diretos ou indiretos decorrentes do uso ou impossibilidade de uso
  • Perda de dados por falha do navegador ou hardware do usuário
  • Decisões tomadas baseadas nas informações do sistema
  • Erros ou imprecisões nos dados importados de fontes externas

Suspensão e Término

Podemos Suspender ou Terminar seu Acesso se:

  • Você violar qualquer termo deste acordo
  • Detectarmos uso inadequado ou suspeito
  • Houver ordem administrativa ou legal
  • Você deixar de ser funcionário da SEFAZ-SE
  • Por razões de segurança ou manutenção

Você Pode Cessar o Uso a Qualquer Momento

Simplesmente pare de acessar o sistema. Recomendamos limpar o cache do navegador para remover dados locais.

Conformidade com Microsoft Platform

Este sistema utiliza a Microsoft Identity Platform e está em conformidade com o Microsoft Platform Agreement.

Nossos Compromissos

  • Não usar APIs da Microsoft para fins proibidos
  • Não construir bases de dados extensas sem justificativa
  • Não usar dados para marketing sem consentimento
  • Manter segurança adequada de credenciais e tokens
  • Respeitar limites de uso e chamadas de API
  • Atualizar informações de registro quando necessário

Modificações nos Termos

Reservamo-nos o direito de modificar estes termos a qualquer momento. Quando fizermos alterações significativas:

  • Atualizaremos a data "Última atualização" no topo desta página

  • Publicaremos versão atualizada neste endereço

O uso continuado após alterações implica aceitação dos novos termos.

Lei Aplicável e Jurisdição

Estes termos são regidos pelas leis brasileiras. Quaisquer disputas serão resolvidas no foro da comarca de Aracaju/SE.
